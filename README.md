# VAT Registration RAG Chatbot (Next.js + FastAPI)

ระบบสืบค้นข้อกฎหมายและระเบียบปฏิบัติเกี่ยวกับการจดทะเบียนภาษีมูลค่าเพิ่ม (VAT Registration) สำหรับเจ้าหน้าที่กรมสรรพากร 
โครงการศึกษาอิสระ (IS) ระดับปริญญาโท คณะสถิติประยุกต์ NIDA

ระบบได้รับการออกแบบโครงสร้างแยกส่วน (Decoupled Architecture) และทำงานเป็น **ระบบปิดภายในเครื่อง 100% (On-Premise / 100% Local RAG)** เพื่อรักษาความมั่นคงปลอดภัยสูงสุดของข้อมูลทางราชการ

## ขั้นตอนการติดตั้งและรันระบบ (Setup & Running Guide)

เนื่องจากระบบนี้เป็นแบบทำงานบนเครื่องโลคอล 100% ท่านจำเป็นต้องติดตั้ง **Python** และ **Node.js** ลงในเครื่อง Windows ก่อนเริ่มใช้งาน

### 1. ติดตั้งซอฟต์แวร์ที่จำเป็น (Prerequisites)

#### 1.1 การติดตั้ง Python (สำหรับ Backend)
1. ดาวน์โหลด Python เวอร์ชั่น 3.10 ขึ้นไปได้ที่ [python.org](https://www.python.org/downloads/)
2. **สำคัญ:** ในขั้นตอนติดตั้ง ให้ทำเครื่องหมายถูกที่ช่อง **"Add Python to PATH"** หรือ **"Add python.exe to PATH"** ด้านล่างสุด ก่อนกด Install
3. หากติดตั้งแล้ว ให้ปิดและเปิด Command Prompt ใหม่ แล้วพิมพ์ `python --version` เพื่อทดสอบ

#### 1.2 การติดตั้ง Node.js (สำหรับ Frontend)
1. ดาวน์โหลด Node.js (แนะนำเวอร์ชั่น LTS) ได้ที่ [nodejs.org](https://nodejs.org/)
2. ดำเนินการติดตั้งตามตัวเลือกดีฟอลต์ทั้งหมด
3. พิมพ์ `node -v` และ `npm -v` ใน Command Prompt เพื่อทดสอบความถูกต้อง

---

### 2. วิธีการเริ่มใช้งานระบบ (How to Run)

ระบบได้เตรียมสคริปต์สั้นรันอัตโนมัติ (.bat) เพื่อความสะดวกรวดเร็วในการทำงานจริง ดังนี้:

#### 2.1 รันระบบหลังบ้าน (Backend - FastAPI)
1. เปิดโฟลเดอร์ `backend/`
2. ดับเบิ้ลคลิกไฟล์ `run.bat`
3. สคริปต์จะทำการติดตั้งแพ็คเกจผ่าน `pip` และรันเซิร์ฟเวอร์หลังบ้านโดยอัตโนมัติบนพอร์ต 8000 (`http://127.0.0.1:8000`)

#### 2.2 รันระบบหน้าบ้าน (Frontend - Next.js)
1. เปิดโฟลเดอร์ `frontend/`
2. ดับเบิ้ลคลิกไฟล์ `run.bat`
3. สคริปต์จะทำการติดตั้งแพ็คเกจผ่าน `npm install` และรันเว็บเซิร์ฟเวอร์ Next.js พัฒนาบนพอร์ต 3000 (`http://localhost:3000`)
4. เมื่อรันสำเร็จ ให้เปิดเว็บเบราว์เซอร์ไปที่: **`http://localhost:3000`**

## รายละเอียดฟีเจอร์สำคัญของระบบ (Key Features & Implementation)

1. **การจำประวัติและเกลาคำถาม (Query Re-writing)**: ระบบจะส่งประวัติการสนทนาของเซสชันปัจจุบันกลับไปพร้อมกับคำถามใหม่ หากตรวจพบว่าผู้ใช้พิมพ์ถามต่อเนื่อง เช่น *"ต้องยื่นจดที่ไหน"* หลังจากถามคำถามเรื่อง *"การจดทะเบียน VAT"* ระบบจะเกลาคำถามอัตโนมัติเป็น *"ช่องทางและสถานที่ในการจดทะเบียนภาษีมูลค่าเพิ่ม (VAT)"* ก่อนนำไปสืบค้นใน Vector Store
2. **ระบบดักจับคำถามนอกขอบเขต (Out-of-Scope Guardrails)**: หากผู้ใช้พิมพ์ถามในประเด็นที่ไม่ได้เกี่ยวกับการจดทะเบียนภาษีมูลค่าเพิ่มของกรมสรรพากร เช่น *"ขอคืนภาษีท่องเที่ยวประชารัฐ"* หรือ *"สมัครงาน"* ระบบจะดักจับและส่งการแจ้งเตือนปฏิเสธพร้อมแนะแนวทางที่ถูกต้องทันที
3. **ระบบดักจับความขัดแย้งของกฎระเบียบ (Conflict Detection)**: ตรวจจับประเด็นขัดแย้งเชิงปฏิบัติ เช่น รายได้ไม่ถึง 1.8 ล้านบาทต่อปีแต่อยากจดทะเบียน VAT หรือ การยื่นจดทะเบียนเกินกว่าระยะเวลา 30 วันที่กฎหมายกำหนด เพื่อแสดงสัญลักษณ์เตือนความเสี่ยง (⚠️) ให้เจ้าหน้าที่ทราบ
4. **ฐานข้อมูลเก็บบันทึกประวัติ (SQLite Database Logs)**: ทุกการมีปฏิสัมพันธ์จะถูกเก็บบันทึกลงฐานข้อมูล `chatbot_logs.db` อัตโนมัติ เพื่อนำไปคำนวณสถิติแสดงผลบนแผงแดชบอร์ดผู้ดูแลระบบ (Admin Panel)
5. **ปุ่มปรับขนาดตัวอักษรหน้าเว็บ (Accessibility Size Control)**: ปรับขนาดตัวอักษรของกล่องข้อความสนทนาและตัวบทกฎหมายดิบได้ 4 ระดับ (A- / A+) บนมุมขวาของหน้าจอหลักทันที
